# IMC Department Location Finder

A web based platform designed to help citizens quickly locate the appropriate department within Indore Municipal Corporation (IMC) based on their query or service requirement. The platform simplifies navigation by guiding users to the correct department, reducing confusion and improving accessibility to municipal services.

## Features

- Department search and discovery
- Department wise information and descriptions
- User friendly interface for easy navigation
- Fast and responsive design
- Mobile friendly experience
- Optimized performance with Vite
- Deployed on Vercel

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Vercel
